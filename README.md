# Library Console App 

A simple console application written in C# for managing a basic library catalog. This project was created to practice working with 2D arrays, loops, and basic conditional logic.

## Features
Location Search:** Find an author by specifying their shelf and spot numbers.
Author Search:** Find the exact location (shelf and spot) of a book by entering the author's name (case-insensitive search).
Interactive UI:** Simple and easy-to-use console menu loop.
